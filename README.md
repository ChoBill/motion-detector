# motion detect
Using the OpenCV 2 library to implement the motion detector.
This program capture images from camera and analysis the images.
If motion detected, the alarm will be triggered.

# Quick Start
> Requirements:
> * python3-opencv
> * aplay

Run the motion detector:
```
python3 motion.py 
```

Exit the program
```
Press 'ESC'
```
