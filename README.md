# motion detect
Using the OpenCV 2 library to implement the motion detector.
This program capture images from camera and analysis the images.
If motion detected, the alarm will be displayed.

# Quick Start
> Requirements:
> * python-opencv

Run the test server:
```
python motion.py 
```
